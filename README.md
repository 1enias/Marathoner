# Marathoner
An automatic python script for the Marathoner Pin in Bits &amp; Bops
## WARNING: THIS IS A CHEAT APPROACH 
For those who have trouble with the Marathoner pin and/or suffer from arthritis as I do, and do **NOT MIND CHEATING**, here is the python script I created to get this pin.
In order for this to work, you will have to start the marathon minigame in **multiplayer mode** (where you need to press both space and left click in the get-ready phase).
After you are prompted to take your first step, press the default F9 key for the script to kick in.
When you have traveled around the globe and visited the starting point again, press ESC to kill the script.
Return to the main lobby, and the pin should be yours.
Have fun.
